# jjson-api

JAVA-Spingboot example with
    - api
    - autogenerated swagger
    - store in json-file

## Env-Variables
- DATA_DIR
- DATA_FILE
 
## Example 
```
    export DATA_DIR=./test/data
    export DATA_FILE=data-entries.json
    export LOG_LEVEL=TRACE
    java -jar target/jjson-api.jar --logging.level.de.mgeo=${LOG_LEVEL}
```
