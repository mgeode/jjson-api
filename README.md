# jjson-api

JAVA-Spingboot example with
    - api
    - autogenerated swagger
    - store in json-file

## Env-Variables
- DATA_DIR
- DATA_FILE
 
## Example 
```
    export DATA_DIR=/data
    export DATA_FILE=production.json
    export LOG_LEVEL=DEBUG
    java -jar jjson-api.jar --logging.level.de.mgeo=${LOG_LEVEL}
```
